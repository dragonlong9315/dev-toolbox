# Dev工具箱 — 免费在线开发者工具

一个纯前端的在线工具集合，托管在 GitHub Pages（完全免费）。

## 包含工具

1. JSON 格式化/校验
2. URL 编解码
3. Base64 编解码
4. 时间戳转换
5. 正则表达式测试
6. MD5/SHA 哈希生成
7. 二维码生成
8. 颜色格式转换 (HEX/RGB/HSL)
9. 文本 Diff 对比
10. Markdown 预览

## 特点

- 🚀 纯前端运行，数据不上传
- 📱 响应式设计，手机可用
- 💰 完全免费，零成本部署
- 🔓 开源，可自行修改

## 部署

1. Fork 此仓库
2. 在 Settings → Pages 中启用 GitHub Pages（Source: main 分支）
3. 访问 `https://你的用户名.github.io/dev-toolbox/`

或直接运行 `bash deploy.sh`

## 本地开发

直接用浏览器打开 `public/index.html` 即可。
